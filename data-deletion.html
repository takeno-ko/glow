EN — How to delete your data

All Glow user data (pimple logs, daily care logs, character preferences, nickname, friend codes, gift records, etc.) is stored locally on your device only. We do not store this data on our servers.

To delete everything immediately:

  1. Uninstall the Glow app from your device — this removes all local data.
  2. To reset the anonymous advertising ID used by Google AdMob: Android: Settings → Google → Ads → Reset advertising ID / Delete advertising ID.

For analytics, we use PostHog with an anonymous, randomly-generated identifier. To request deletion of any analytics events associated with your installation, email us:

📧 taketa9792@gmail.com
Subject: Glow Data Deletion Request
Please include the date you installed Glow so we can locate the relevant anonymous events.

We respond within 14 days.

JA — データの削除方法

Glow のユーザーデータ (ニキビ記録・生活記録・キャラ設定・ニックネーム・トモダチコード・ギフト履歴など) はすべて あなたの端末内のみに保存 されます。当方のサーバーには記録されません。

すぐに削除する方法:

  1. Glow アプリをアンインストール → ローカルデータがすべて削除されます
  2. Google AdMob の匿名広告 ID を初期化したい場合: Android: 設定 → Google → 広告 → 広告 ID をリセット / 広告 ID を削除

匿名分析 (PostHog) のイベント削除を希望される場合は、下記メールアドレスにご連絡ください:

📧 taketa9792@gmail.com
件名: Glow Data Deletion Request
Glow をインストールされた日付を併記してください (該当する匿名イベントの特定に必要です)。

14 日以内に返信します。
